## 🔧 Installation Guide  
1. **Minecraft Bedrock**:  
   - Download the `.mcpack` file  
   - Open it with Minecraft  
   - Apply it from Settings > Global Resources  

2. **Minecraft Pe**:  
   - Download the `.mcpack` file  
   - Open **Z archivers**.  
   -  Click on the **file** Press arrow and select **minecraft**.
